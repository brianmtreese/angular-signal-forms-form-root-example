# Angular formRoot Directive Demo

This project demonstrates the new `formRoot` directive in Angular's signals-based forms API. It showcases how to use the directive to simplify form submission handling and improve the developer experience when working with reactive forms.

## Overview

This demo includes a signup form with:
- **Client-side validation** using `required` and `minLength` validators
- **Form state management** using Angular signals
- **The `formRoot` directive** for declarative form submission

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Running the Application

```bash
npm start
```

Navigate to `http://localhost:4200` in your browser.

## Angular Version

This project uses **Angular 21.2.0-next.3** which includes the new signals-based forms API and `formRoot` directive.
